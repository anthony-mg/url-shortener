## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites
* bun
  
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/anthony-mg/url-shorterner.git
   ```
2. Install NPM packages
   ```sh
   cd url-shorterner
   ```
   ```sh
   bun install
   ```
4. Start the dev environment
    ```sh
    bun run dev
    ```
5. To go localhost:8080
